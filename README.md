PROJECT NAME: Akan Name Generator

PROJECT DESCRIPTION: The Akan Name Generator is a web application that calculates your Akan name based on your birthday and gender. Akan names are derived from Ghanaian culture, where children are traditionally given their first name based on the day of the week they were born.

AUTHOR: Fiona Njeri Mbugua

SETUP INSTRUCTIONS
1. Clone this repository
2. Open index.html in your web browser
3. Enter your birth date and select your gender
4. Click submit to see your Akan name and day of birth

BDD(Behavior-Driven Development)
